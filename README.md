# PintorApp Lite

Aplicación PWA para pintores profesionales. Versión simplificada de PintorApp Pro.

## URL

`https://bufeteimpagosojuridicos-byte.github.io/pintorapp-lite/`

## Funciones

- Presupuestos con PDF y envío por WhatsApp
- Carta de colores NCS + RAL + RGB
- Calculadora de superficies por tramos
- Dictado por voz
- IGIC 7% / IVA 21%
- Datos guardados en el móvil

## Uso

Abre `index.html` en el móvil o escaneando el QR de la URL.
